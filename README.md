## 📌 Project Overview
This project demonstrates how to display digital numbers (0–9) using a Seven Segment Display controlled by an Arduino. It is a beginner-friendly embedded systems project that helps understand GPIO pin control and digital output.

## 🎯 Objective
Display numbers from 0 to 9 on a Seven Segment Display.
Learn digital output interfacing with Arduino.
Understand how LEDs inside a Seven Segment Display work.

## 🛠 Components Required
Arduino Uno/Nano
Seven Segment Display (Common Cathode)
220Ω Resistors (7)
Breadboard
Jumper Wires
USB Cable

## ⚙ Working Principle
Each segment (A–G) is controlled by an Arduino digital pin.
The Arduino turns ON specific segments to display numbers from 0 to 9.
Each number is displayed for a short delay before changing to the next.

## ▶ How to Run
Connect the Seven Segment Display to the Arduino.
Upload the Arduino code.
Power the Arduino using USB.
The display will continuously count from 0 to 9.
